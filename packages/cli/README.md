# zskills CLI

ZSkills command-line client.

## Commands

- `zskills add <owner/repo> [--skill <name>] [-a <agent>] [--global]`
- `zskills list`
- `zskills find <query> [--repo <owner/repo>]`
- `zskills check`
- `zskills update`
- `zskills remove [skill] [--all] [-a <agent>]`
- `zskills init <name>`

## Build

npm run build -w zskills
