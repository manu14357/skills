# askills CLI

ASkills command-line client.

## Commands

- `askills add <owner/repo> [--skill <name>] [-a <agent>] [--global]`
- `askills list`
- `askills find <query> [--repo <owner/repo>]`
- `askills check`
- `askills update`
- `askills remove [skill] [--all] [-a <agent>]`
- `askills init <name>`

## Build

npm run build -w askills
