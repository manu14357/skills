# Canvas Design

## Use This Skill When

- Creating original visual art, posters, or design pieces
- User requests visual design without specific templates
- Designing with a particular aesthetic or philosophical approach
- Creating artwork for albums, events, publications, or galleries
- Building visual identities or design systems
- Crafting sophisticated, exhibition-quality graphics

## When NOT to Use

- User wants existing artwork recreated or copied
- Generic document decoration or template application
- When copyright or originality concerns exist
- Non-visual design tasks (copywriting, layout without artistry)
- User explicitly requests simple, amateur designs

## Context: Design Mastery

**Undeveloped**: Generic graphics, obvious templating, amateur execution, heavy text-reliant design.

**Target**: Original visual philosophy expressed through expert composition, minimal text, sophisticated aesthetic movement.

**Optimized**: Museum-quality artwork with masterful craftsmanship, perfect execution, refined aesthetic vision, professional presentation.

## Core Principle

Exceptional visual design communicates through **form, space, color, and composition**—not text. Art objects are created through a deliberate aesthetic philosophy executed with painstaking craftsmanship, appearing as though produced by someone at the absolute top of their field.

## Instructions

### Phase 1: Create Design Philosophy

**Step 1: Name the Movement**
- Create a 1-2 word aesthetic movement name
- Examples: "Brutalist Joy", "Chromatic Silence", "Metabolist Dreams"
- Should evoke a visual worldview, not a literal concept

**Step 2: Write the Philosophy (4-6 paragraphs)**

Express the aesthetic through these visual dimensions:

| Dimension | Questions to Address |
|-----------|----------------------|
| **Space & Form** | How does empty space function? What geometric or organic forms define the movement? |
| **Color & Material** | What color palette? How do materials feel? Matte vs. glossy? Tactile qualities? |
| **Scale & Rhythm** | What's the scale relationship between elements? Repetition patterns? Cadence? |
| **Composition & Balance** | Symmetrical or asymmetrical? Central focus or distributed? Grid-based or organic? |
| **Visual Hierarchy** | How does the eye move? What's emphasized? What recedes? |

**Critical Requirements:**
- Avoid repeating the same points about color, space, or type
- **REPEAT EMPHASIS** on expert craftsmanship: "meticulously crafted," "painstaking attention," "master-level execution," "countless hours of labor"
- Remain specific enough to guide visual direction, but leave creative interpretation room
- Philosophy should guide future visual expression, not constrain it
- NO LITERAL REFERENCES—purely aesthetic language

**Philosophy Examples:**

*Concrete Poetry*: Monumental form and bold geometry. Massive color blocks, sculptural typography (giant single words, tiny labels), Brutalist spatial divisions. Ideas expressed through visual weight and spatial tension. Text as rare, powerful gesture—only essential words integrated into visual architecture. Every element placed with master craftsman precision.

*Chromatic Language*: Color as primary information system. Geometric precision where color zones create meaning. Minimal typography—small sans-serif labels. Josef Albers meets data visualization. Information encoded spatially and chromatically. Painstaking chromatic calibration throughout.

*Analog Meditation*: Quiet visual contemplation through texture and breathing room. Paper grain, ink bleeds, vast negative space. Photography and illustration dominate. Typography whispered (small, restrained). Japanese photobook aesthetic. Images breathe across pages. Balanced with meditative care.

### Phase 2: Deduce the Subtle Conceptual Thread

**The Invisible DNA:**
- Identify subtle, refined conceptual references embedded in the art
- NOT literal or obvious—sophisticated and intuitive
- Familiar audiences sense it without direct reference
- Works like a jazz musician quoting another song—only cognoscenti catch it
- Enhances depth without announcing itself

### Phase 3: Create the Canvas

**Execution Principles:**

| Principle | Implementation |
|-----------|-----------------|
| **Visual-First** | 90% design, 10% essential text |
| **Minimal Text** | Only contextual labels or key phrases |
| **Perfect Spacing** | No overlaps, proper margins, breathing room |
| **Sophisticated Typography** | Design-forward fonts; text as visual element |
| **Masterful Craftsmanship** | Appears meticulously crafted, labored over, expert-level |
| **Single-Page Clarity** | One cohesive, downloadable PDF or PNG (unless multi-page requested) |
| **Professional Margins** | All elements contained within boundaries with clear separation |

**Design Decision Framework:**

1. Start with design philosophy as foundation
2. Apply visual language of systematic observation
3. Use repeating patterns and perfect shapes
4. Build meaning through patient repetition
5. Employ sparse, clinical typography as contextual element
6. Select intentional, cohesive limited color palette
7. Position key phrases/details subtly
8. Treat composition with scientific rigor

**Text Handling:**
- Context determines scale (punk poster ≠ minimalist studio identity)
- Most fonts should be thin/elegant
- Always design-forward and visually prioritized
- Search `./canvas-fonts` for sophisticated typefaces
- Integrate typography as part of the art itself
- Never overlaps or off-page elements

**Craftsmanship Checklist:**
- ✓ Composition appears labored over with care
- ✓ Spacing/alignment screams expert execution
- ✓ Color choices feel intentional and refined
- ✓ Typography serves the visual, not the text
- ✓ Every detail perfect, nothing amateur
- ✓ Looks worthy of museum or magazine feature
- ✓ Could be shown to prove expertise

### Phase 4: Multi-Page Option (if requested)

- Create unique pages that reference original philosophy distinctly
- Each page tells part of a larger visual story
- Treat as pages in a coffee-table book
- Maintain cohesive aesthetic across pages
- Exercise full creative freedom within philosophy boundaries

## Output

Deliver:

1. **Design Philosophy (MD file)**
   - 4-6 paragraph manifesto of aesthetic movement
   - Visual expression guidance
   - Craftsmanship emphasis

2. **Canvas Artwork (PDF or PNG)**
   - Single-page or multi-page depending on request
   - Museum-quality execution
   - Perfect spacing and formatting
   - Minimal, sophisticated typography
   - Original visual creation, never copied

3. **Execution Summary** (brief, if appropriate)
   - Philosophy applied
   - Aesthetic movement demonstrated
   - Key visual decisions made
   - Conceptual thread embedded
